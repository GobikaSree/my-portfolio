Step 1: Document title
Static Portfolio Website Hosting using GitHub Pages

Step 2: Objective
To create and host a static portfolio website using HTML and CSS
and deploy it using GitHub Pages with HTTPS enabled.

Step 3: Tools / Technologies Used
- HTML
- CSS
- GitHub
- GitHub Pages

Step 4: Step-by-step Procedure
1. Created a static website using HTML and CSS.
2. Tested the website locally using a web browser.
3. Created a new public repository in GitHub.
4. Uploaded index.html and styles.css files to the repository.
5. Enabled GitHub Pages from repository settings.
6. Deployed the website using the main branch.
7. Accessed the live website using the generated URL.

Step 5: Live URL
https://gobikasree.github.io/my-portfolio/

Step 6: Testing
- Verified website accessibility using the live URL.
- Checked HTTPS security.
- Confirmed updates reflected after committing changes.

Step 7: Result
The static portfolio website was successfully deployed and is
accessible publicly with HTTPS enabled.

Step 8: Conclusion
This project helped in understanding static website hosting,
version control using GitHub, and deployment using GitHub Pages.
